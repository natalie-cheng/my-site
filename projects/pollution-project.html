Analysis & Prediction of US Pollution Trends

October 2021 | CoderSchool

My Analysis & Prediction of US Pollution Trends project was my Capstone project as part of the Data Science in Python course with CoderSchool. View the Github repository here.

#csharp | #unity | #game-design | #project

This project was the first application of the data cleaning and analysis skills I learned throughout the course. I ran this program in a conda envionrment on my laptop, and ultimately transferred it onto a streamlit webpage for presentation (the page is no longer available).

I used publicly sourced data on pollutant levels in the US and analyzed the effects of 4 different air pollutants (NO2, SO2, O3, CO) from 2010 to 2016, with an emphasis on California. I utilized my data cleaning skills to process the dataset.

My final code and streamlit webpage displayed graphs that plotted pollution change, with the user able to select either the time frame (yearly, monthly), the location (the US, California), or the pollutant.

Then I created a graph that ranked the 50 states' levels of pollution, given a particular pollutant (user-chosen).

I also utilized machine learning to produce a prediction model on the levels of each pollutant from 2016 to 2026, both for the US and specifically California.

Finally, I used a publicly sourced geo data file to produce a map of the US with corresponding state colors based on the level of a particular pollutant (similar to a weather map).
