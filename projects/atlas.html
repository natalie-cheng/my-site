Atlas

January 2024 - March 2024 | COMP_SCI 377-0: Game Design Studio

Atlas is a 2D game I designed and built as a 3-month project for my COMP_SCI 377-0 Game Design Studio project. Working in a team of 4, we produced a fun, multi-level adventure game inspired by Greek Mythology. View the Github repository here.

#csharp | #unity | #adventure | #team | #game-design

Atlas, a Greek hero, has just successfully stolen the Golden Fleece! Unfortunately, he had to leave in a hurry, and heâ€™s now stranded on an island in the middle of nowhere with no way to get home. Play as Atlas, surviving at sea, fighting through mythological monsters, and escaping angry gods to safely bring the Golden Fleece back home.

This project was a great experience in organizing and designing a larger-scale game project. We underwent a shortened version of the game design cycle, including concept, prototyping and user testing. I particularly enjoyed how this project intersected my passion for game design and Classics.

Features

  • 5 minigame levels.
  • 5+ unique enemies.
  • Animated player movement and enemy movement/attack.
  • Narrative cutscenes.
