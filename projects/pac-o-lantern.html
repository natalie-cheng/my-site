Pac-O-Lantern

October 2023 | COMP_SCI 376-0: Game Design and Development

Pac-O-Lantern is a 2D Arcade Game that I built as a weekly assignment for my COMP_SCI 376-0 Game Design class. It is a halloween-themed game loosely based on Pac-Man. View the Github repository here.

#csharp | #unity | #game-design | #project

The player is a pumpkin, whose goal is to collect all the candies on the map. However, ghosts will randomly spawn and chase after you â€“ you can either try to avoid them, or shoot them will fireballs! You can win by collecting all the candies, or lose by losing all three lives being hit by ghosts.

This was the first freestyle project assignment we had in COMP_SCI 376. I really enjoyed having more freedom and choice in the UI and gameplay design. I learned how to utilize audio and many more UI elements, as well as game mechanics such as scene management and so on.

Features

  • Life-based gameplay system.
  • Enemy mob auto-tracking and movement.
  • Animated fireball projectiles.
  • Dynamic score tracking.
  • Background music and event-based sound effects.
